App Mock-Ups

Exercise App

In this project, the objective was to create an exercise app with multiple screens as well as navigations through those screens. In the video you can see how the user can navigate from the Home screen to a repetitive execise or 'Push-Ups' screen. From there, the user can also navigate to a 'suggested' excerise or the duration exercise screen. This project really made me think about the best ways to seamlessly connect screens through different buttons!

Quiz App

In this'Quiz App' project, I was tasked with setting up three different multiple choice questions, one even having two right answers, and then presenting the user with an answer key at the end. This project required me to really think how I could smoothly transition from one question to the next. It also helped me learn how to present users with choices in which they could actively interact with.

To-Do App

This 'To-Do App' required a set up in which users can check off which tasks they have and haven't done, as well as the option to add a new task to the list. This project implemented the use of React Native to make it work. I found this project a bit more difficult as getting the task to appear completed was a bit of a hurdle for me. In the video you can see the user adding two more tasks to the list. I was able to learn the ways in which React Native can help in the making in these types of apps!

UI Mock-Ups

Exercise App Mock-Up

Here we have an exercise app mock-up that I designed. I implemented the design of three different screens. The first being a recording feature that the user could stop and start at their will. The second being a duration timer/distance tracker. And lastly, the third screen presenting the user with a recommended exercise based on their information.

To-Do App Mock-Up

In this design, I was tasked with creating a 'To-Do' app. Here I also designed 3 different screens, a Home, Tasks, and Profile screen. In 'Home', the user is able to add a new task as well as a description to go along with said task and then a date to complete it by. Moving on to the 'Tasks' screen, the user is able to see all of the tasks they have or have not completed, check them off, and even filter by tasks they haven't completed. In the 'Profile' screen, the user is able to set their profile picture up and input their information.

New Pinterest Mock-Up

In this exercise, I was asked to take an existing application and redesign it so that it would implement features I thought would add to the app. I chose to tweak Pinterest since I find myself going on the app frequently. I first wanted to spruce up the navigation bar at the bottom because I thought the one in the app was a bit boring, so I added little touches like the heart home icon and the big plus button to get the users' attention. Then I added a stories feature at the top. I thought it would be fun if people could add pinterest pictures to a story just to see what everyone is looking at and enjoying on the app if they so desire to share. Lastly, I added a recommended page based on the users' searches.
